# ⚽ SportingRank

**SportingRank** is a professional, full-stack sports rankings platform. It provides dynamic, up-to-date global rankings for the top 10 sports in the world.

## ✨ Key Features
- **Dynamic Rankings**: Real-time standings for 10 major global sports.
- **Interactive Voting**: Polling system to vote for the best teams.
- **Admin Panel**: Secure dashboard to manage sports, teams, and site settings.
- **SEO Optimized**: Built to rank for keywords like "Sport Rank".
- **Responsive Design**: Modern, dark arena aesthetic that works on all devices.
- **Live Search**: Instantly find teams or sports from the navbar.

## 🛠️ Technology Stack
- **Frontend**: Tailwind CSS, CountUp.js, Chart.js.
- **Backend**: PHP 8.x.
- **Database**: MySQL (or SQLite for development).
- **Server**: cPanel / Apache.
